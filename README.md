# Cycle Store (Frontend Demo)

This is a **frontend-only demo website** for a cycle store, built using **HTML, CSS, and Vanilla JavaScript**.

There is **no backend** and **no real payment system**.  
All user data is stored in the browser using **localStorage**.

---

## Features

- Responsive modern UI
- User registration and login (localStorage based)
- Login / Logout functionality
- Dynamic header and footer
- Product listing page
- Product details page
- Checkout demo page with warning message
- Protected profile page

---

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
- Browser localStorage
